### 4. 리스트 조작

#### 리스트 요소 수정

```python
a = [1,2,3,4,5]
a[2] = a[2]*10
print(a)
-------------------------
[1,2,30,4,5]
```



#### 리스트 요소 삭제

```python
a = [1,2,3,4,5]
del a[2:]
a
-----------------
[1,2]
```



#### 리스트 요소 추가

```python
a = [1,2,3,4,5]
a.append(6)
a
-------------------
[1,2,3,4,5,6]
```



#### 리스트 정렬

```python
a = [1,4,3,2]
a.sort()
a
----------------
[1,2,3,4]
```

```python
a = [1,4,3,2]
a.sort(reverse=True)
a
-----------------
[4,3,2,1]
```



#### 리스트 뒤집기

reverse를 이용하면 리스트를 뒤집어준다.

이를 이용하여 역순 정렬을 만들어보면?

```python
a = [1,4,3,2]
a.sort()
a.reverse()
a
----------------------
[4,3,2,1]
```

#### 리스트 확장

리스트에 리스트를 추가하고 싶은 경우

리스트에 요소를 추가하는 append와 헷갈리지 말것

```python
a = [1,2,3]
b = [4,5]
a.extend(b)
a
-----------------------
[1,2,3,4,5]
```

위와 동일한 방식으로 아래와 같이 연산자로 할 수 있다.

```python
a = [1,2,3]
b = [4,5]
a += b
a
-----------------------
[1,2,3,4,5]
```



### 5. 리스트 연산

####  리스트 요소 존재 여부 확인

```python
a = [1,2,3,"test","aaa"]

---------------------------

```

#### 리스트 요소 위치 찾기

```python
a = [1,2,3,"test","aaa"]
a.index("test")
---------------------------

```
